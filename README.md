# Bingo GitHub

## Création du repo

Créer un repository GitHub public.

Exemple :

https://github.com/MONUSER/bingo

---

## Ajouter les fichiers

Copier tous les fichiers du projet.

---

## Créer un token GitHub

GitHub → Settings → Developer settings → Personal access tokens.

Créer un token avec :

- repo

---

## Héberger avec GitHub Pages

Settings → Pages

Branch:

- main
- /(root)

Puis Save.

Le site sera disponible sur :

https://MONUSER.github.io/bingo/

---

## Utilisation

### Affichage public

/index.html

### Admin mobile

/admin.html

Entrer :

- token GitHub
- user GitHub
- nom du repo

Puis envoyer les numéros.
